  • Maze Writer

    A maze generator that can write things!

    Uses the depth-first search algorithm with a recursive backtracker.

    Simply put, it wanders around until it gets stuck, at which point it continues wandering from the last point it wasn't stuck at. When no such point exists the maze is complete.

    Quite pleasing to look at, I'd say.
